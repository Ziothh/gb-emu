# Hardware
## CPU
 - DMG-01 (~= ZiLOG Z80)
 - Clock: 4.16MHz (acts like 1MHz)
 - 16-bit

 - Registers (memory cells, directly integrated in the CPU)
    - 8-bit
        - A F(lags)
        - B C
        - D E
        - H L
    - 16-bit
        - SP (Stack Pointer)
        - PC (Program Counter)


## Memory
 - 8kb work RAM
 - 8kb VRAM
